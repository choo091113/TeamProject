코딩, 멀게만 느껴지고 막상 배우자니 엄두가 나지 않았다면?
나만의 웹사이트를 직접 만들어보며
웹개발의 A to Z를 이해해보세요.
일상 속으로 훌쩍 다가온 IT를 이해하고 미래를 대비할 수 있어요.
더 이상 개발자의 말이 어렵게 들리지 않아요. 자유롭게 소통할 수 있어요.
개발에 대한 이해를 바탕으로, IT 회사 이직을 준비할 수 있어요.


핵심 스킬 3가지
1
코딩 입문이 되는 기본적인 언어
JavaScript, Python 등 말로만 들었던 코딩의 기본 언어를 배워 개발자와 소통하고, 요구사항을 정확히 전달해요.
2
내 손으로 직접 만드는 웹사이트
개발 경험이 없어도 HTML과 CSS, Python으로 웹사이트 구성 요소인 프론트엔드와 백엔드를 구축할 수 있어요.
3
직접 만든 웹사이트를 인터넷에 공개까지
실제로 배포하고 주소만 공유하면, 누구나 내 웹사이트에 자유롭게 방문할 수 있어요.


사용 기술
HTML, CSS, Javascript, jQuery, Fetch, Python, Flask
구현 기능
웹 뼈대를 만들고 꾸미기
웹의 데이터와 뼈대 조직하기
JSON 데이터 요청하기
웹 데이터 스크래핑
등록, 조회 API 만들기
웹 배포하기
